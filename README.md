# Prodesk-IT-MISSION-3
# 📌 Project Description
#Hi there! This project is a GitHub User Search App. this app will connect to the real GitHub Database, fetch data about any developer in the world, and display it on your screen.
-- No modern application lives in isolation. Instagram fetches photos, Amazon fetches products, Uber fetches location. here we are created GitHub user search.

## 🚀 Features
- just search with github user id for the user details
- a simple search bar for searching option.
- we can get details like Name,Bio,DOJ, and resposries links
- we can open latest top 5 repositries of user in a new tab
## Used Technologies
- HTML(hypertext markup language)
- CSS(Cascaded style sheet)
- javascript
